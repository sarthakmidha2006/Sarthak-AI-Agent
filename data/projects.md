## AI Persona Agent

Type: AI Engineering Project

Status: In Progress

### Overview

AI Persona Agent is an end-to-end AI system designed to represent my professional background through natural conversation across chat and voice interfaces. The system is capable of answering questions about my experience, projects, skills, and career goals while remaining grounded on verified information from my resume, portfolio, GitHub repositories, and project documentation.

The project was built to explore real-world AI engineering concepts including Retrieval-Augmented Generation (RAG), tool calling, evaluation frameworks, conversational AI, and AI agent architectures.

### Problem

Traditional resumes are static and limit how recruiters explore a candidate's experience. Most AI chatbots also struggle with hallucinations and often provide inaccurate responses when discussing personal information.

The challenge was to create an AI system that could answer questions accurately, remain grounded on verified information, support natural conversations, and automate interview scheduling.

### Solution

Built a shared-brain architecture where both chat and voice interfaces use the same retrieval and reasoning system.

The system ingests:

* Resume
* Portfolio content
* GitHub repositories
* Project documentation
* Commit history

The retrieved information is used to generate grounded responses while minimizing hallucinations.

### Features

* Retrieval-Augmented Generation
* Resume and GitHub grounding
* Hybrid retrieval
* Tool calling
* Calendar scheduling
* Voice and chat support
* Evaluation framework
* Prompt injection protection

### Tech Stack

* Python
* FastAPI
* ChromaDB
* SQLite
* GitHub API
* Groq/OpenAI APIs
* BM25 Retrieval
* Vector Search

### Key Learnings

* AI Agent Architecture
* RAG Systems
* Tool Calling
* LLM Evaluation
* Prompt Security
* Production AI Workflows
